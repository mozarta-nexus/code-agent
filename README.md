# Code Agent

从零到一实现一个 Claude Code 风格的代码智能体。

## 项目目标

通过学习 Claude Code 的设计理念与实现方式，从零构建一个具备代码理解、编辑、执行等能力的 AI 代码助手。

## 核心功能（规划中）

- 代码理解与搜索
- 代码编辑与重构
- 命令行交互界面
- 工具调用与编排
- Git 集成
- 上下文管理

## 技术栈

- Python

## 快速开始

```bash
# 克隆项目
git clone https://github.com/mozarta-nexus/code-agent.git
cd code-agent

# 安装依赖
pip install -r requirements.txt
```
